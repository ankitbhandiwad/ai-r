# AI-R

Next.js App Router port of the Michigan Thumb live air traffic viewer.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

Optional environment variables:

- `OPENSKY_CLIENT_ID`
- `OPENSKY_CLIENT_SECRET`
- `FLIGHT_REFRESH_INTERVAL_MS`
- `REGISTRY_LOOKUPS_PER_REQUEST`
